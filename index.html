Information

First

Add WHATEVER you want in each container. If you want to make the containers bigger, change the p width in CSS. Content will not overflow when you specifty width in these specific containers.

Second

For example, when you set the paragraph width (p) to 300px ALL containers will be 300px.

Third

What about height? The height can be changed in the class container. (.container: height: 300px)

Fourth

Enjoy and feel free to customize this project. Add lists, pictures, animations, etc.
